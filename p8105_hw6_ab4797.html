Problem 1

Reading in Birthweight Data

birthweight = 
  read_csv("./data/birthweight.csv")

Cleaning Birthweight Data

birthweight_clean = 
  birthweight %>% 
  janitor::clean_names() %>% 
  drop_na() %>% 
  mutate(mrace = as.factor(mrace),
         babysex = as.character(babysex),
         frace = as.factor(frace),
         malform = as.character(malform)) %>% 
    mutate(mrace = recode(mrace,
                   '1' = "white", 
                   '2' = "black", 
                   '3' = "asian", 
                   '4' = "puerto rican",
                   '8' = "other")) %>% 
    mutate(frace = recode(frace,
                   '1' = "white", 
                   '2' = "black", 
                   '3' = "asian", 
                   '4' = "puerto rican",
                   '8' = "other",
                   '9' = "unknown"))

Proposing a regression model based on hypothesized structure

fit = lm(bwt ~ mrace + fincome + gaweeks, data = birthweight_clean)

fit %>% 
  broom::tidy() %>% 
  select(term, estimate, p.value) %>% 
  mutate(term = str_replace(term, "mrace", "Mother's Race: ")) %>%

This study found that African-American and Latino mothers who smoked and lived in poorer communities were three times more likely to have an infant with lower birth weight compared to non-smoking and wealthier women.
